HTML, siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto),
es el lenguaje fundamental y estándar para crear la estructura y el contenido
de las páginas web . Se basa en etiquetas o marcas que definen elementos como texto,
imágenes y enlaces, indicando a los navegadores cómo mostrar la información.
HTML trabaja junto con CSS para el diseño y JavaScript para la interactividad, y la versión más reciente y utilizada es HTML5, que incorporó funcionalidades multimedia y gráficos vectoriales. 42
